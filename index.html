What is this anyway?

It's simple really. I bought a domain because I needed a more professional email and my old one was very long to type out. I was forced to use my university email when applying for internships and communicating as a customer which is less than ideal because when I graduate it becomes difficult to direct people to a permanent address.

Also, who is going to get a job with an email address like happyburger69@hotmail.com (not my address thank goodness).

Why this name though?

Simple. it's short which means I can have an address like email@gxi.me and use subdomains like name.gxi.me and it will be just as long as having the domain name.uk.com which is pretty neat.

The gxi doesn't really mean anything. It was originally supposed to be something like g90 since it's easy to remember a number but when I typed it the number made the domain look strange. The memorable domain names have pretty much all been taken by domain speculators looking to get rich and even the domain hacks like grah.am are all taken.

The .me TLD actually is actually country code for Montenegro but the domain is marketed and used like the .name TLD. I chose it because .com is pretty much used up and I don't want to be bound by geographical/national distinction.
